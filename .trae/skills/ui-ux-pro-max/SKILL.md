# UI UX Pro Max

An AI skill that provides design intelligence for building professional UI/UX across multiple platforms and frameworks.

## Features

- **161 Reasoning Rules** for design decisions
- **67 UI Styles** to choose from
- **Design System Generator** - AI-powered reasoning engine
- Multi-platform support (Web, Mobile, Desktop)

## Design System Generation

When user needs a design system, analyze requirements and generate:

### 1. Pattern Selection
- Hero-Centric + Social Proof
- Content-First + Educational
- Product-Centric + Feature Showcase
- Community-Driven + Social Engagement

### 2. Style Definition
Keywords: Soft shadows, subtle depth, calming, premium feel, organic shapes

### 3. Color Palette
```
Primary:    #E8B4B8 (Soft Pink)
Secondary:  #A8D5BA (Sage Green)
CTA:        #D4AF37 (Gold)
Background: #FFF5F5 (Warm White)
Text:       #2D3436 (Charcoal)
```

### 4. Typography
- **Cormorant Garamond** - Elegant, calming, sophisticated
- **Montserrat** - Modern, clean, professional

### 5. Key Effects
- Soft shadows
- Smooth transitions (200-300ms)
- Gentle hover states

## Anti-patterns to Avoid

- Bright neon colors
- Harsh animations
- Dark mode (unless requested)
- AI purple/pink gradients

## Pre-Delivery Checklist

- [ ] No emojis as icons (use SVG: Heroicons/Lucide)
- [ ] cursor-pointer on all clickable elements
- [ ] Hover states with smooth transitions (150-300ms)
- [ ] Light mode: text contrast 4.5:1 minimum
- [ ] Focus states visible for keyboard nav
- [ ] prefers-reduced-motion respected
- [ ] Responsive: 375px, 768px, 1024px, 1440px

## Usage

When user asks for:
- UI design help
- Component styling
- Color scheme recommendations
- Typography advice
- Complete design system
- Landing page design

Invoke this skill to provide professional design intelligence.
